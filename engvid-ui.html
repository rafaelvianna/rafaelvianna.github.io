Redesign of Website

Engvid is a platform where you can learn English by free. But this platform is so old and with too many problems of usability and user experience.

Knowing this I created this fictitious project just for study. I started mapping.

I started by mapping all the pages and recreating the system flow. This done I started in wireframes with the intention of reformulating and redistributing in a more balanced way the elements of each page.

When I finished all the wireframes I started to create the layout.
